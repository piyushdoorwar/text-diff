# text diff

Side-by-side text diff tool with line + character highlights.

## Features
- Two synchronized editors with line numbers
- Added / removed / modified line highlighting
- Character-level highlights for modified lines
- Diff stats (added, removed, modified)
- Copy, paste, download, clear, undo, sample
- Whitespace normalize action

## Usage
Open `index.html` in a browser and paste text into both editors.
